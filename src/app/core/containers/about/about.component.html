Apartment search portal

Description

This application is implemented with Angular and own rxjs State management. Design was made with Angular material, the application offers:

- list of apartments. - search apartment by city - filter apartment by borough. - read more detail about a apartment. - save apartment in state management store as favourite. - highlight saved apartments. - show list of saved apartments.

Purpose

This trial consists of writing a simple angular app:

- full reactive. - Component-First architecture (standalone) - Smart und Dumb Components. - using own rxjs store(state Management). - Design was made with Angular material
