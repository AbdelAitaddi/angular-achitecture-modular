Apartment search portal

Description

This application is implemented with Angular and own rxjs State management. Design was made with Angular material, the application offers:

- list of news. - filter apartment by borough. - search apartment by city - read more detail about a news apartment. - save apartment in state management store. - highlight saved apartments in list of news. - show list of saved apartments.

Purpose

This trial consists of writing a simple angular app:

- full reactive. - modular - component/container architecture. - using own rxjs store.
